# llm-wiki

`llm-wiki` is a local-first, Markdown-first knowledge workspace with a Rust CLI for deterministic wiki maintenance tasks.

## Common Commands

- `llmwiki init`
- `llmwiki install`
- `llmwiki skill install --harness codex --scope repo`
- `llmwiki skill doctor --harness codex --scope repo`
- `llmwiki convert <input>`
- `llmwiki prepare-ingest <path>`
- `llmwiki lint`

## Getting Started

```powershell
llmwiki init
llmwiki install
llmwiki skill install --harness codex --scope repo
```
